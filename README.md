# Partner Hub CN

PWA lista para GitHub Pages con 3 pestañas:

- Partner: dashboard general por tienda, jerarquía, % mujeres/hombres y detalle por partner.
- Aniversarios: filtro por mes, nombre, tienda y años de aniversario.
- Cumpleaños: filtro por mes, nombre, tienda y día de cumpleaños.

## Publicación en GitHub Pages

1. Subir todos los archivos del proyecto al repositorio.
2. Activar GitHub Pages desde `Settings > Pages`.
3. Seleccionar rama `main` y carpeta `/root`.

## Datos esperados

La base fue convertida a `data/partners.js` con los campos:
NUM_EMP, NOMBRE, F_INGRESO, NOM_CCOSTO, TURNO, NOM_PUESTO, REGION, SEXO, F.NAC, DM, MES.
