# Partner Hub CN

PWA lista para GitHub Pages con tres pestañas:

- Partner: dashboard por Región, DM, Tienda, Posición y búsqueda.
- Aniversarios: filtros por Región, DM, Mes/Todos y búsqueda. Exporta/imprime en formato horizontal.
- Cumpleaños: filtros por Región, DM, Mes/Todos y búsqueda. Exporta/imprime en formato horizontal.

Datos principales utilizados: NUM_EMP, NOMBRE, F_INGRESO, NOM_CCOSTO, TURNO, NOM_PUESTO, SEXO, F.NAC, cc, DM y REGION.
